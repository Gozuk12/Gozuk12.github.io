How to install
1. Unzip the folder into:
ChatTriggers/modules/GozukAddons/routes
2. Make sure the folder structure is:
ChatTriggers
↳ Modules
   ↳ GozukAddons
      ↳ Routes
         ↳ <your folder>
            ↳ archer-p5.json
            ↳ archer-terminals.json
            ↳ tank-p5.json
            ↳ tank-terminals.json
3. Run the "/ga select <your folder>" command ingame
